# 문제

정수 4를 1, 2, 3의 합으로 나타내는 방법은 총 7가지가 있다. 합을 나타낼 때는 수를 1개 이상 사용해야 한다.

- 1+1+1+1
- 1+1+2
- 1+2+1
- 2+1+1
- 2+2
- 1+3
- 3+1

정수 n이 주어졌을 때, n을 1, 2, 3의 합으로 나타내는 방법의 수를 구하는 프로그램을 작성하시오.

# 입력

첫째 줄에 테스트 케이스의 개수 T가 주어진다. 각 테스트 케이스는 한 줄로 이루어져 있고, 정수 n이 주어진다. n은 양수이며 11보다 작다.

# 출력

각 테스트 케이스마다, n을 1, 2, 3의 합으로 나타내는 방법의 수를 출력한다.

## 예제 입력 1

```c
3
4
7
10
```

## 예제 출력 1

```c
7
44
274
```

---

# 정답코드

```cpp
#include <iostream>
#include <vector>

using namespace std;

int dp[12];

int main()
{
  
  ios::sync_with_stdio(false);
  cin.tie(NULL);

  int T;
  cin >> T;

  dp[1] = 1; // {1}
  dp[2] = 2; // {1+1, 2}
  dp[3] = 4; // {1+1+1, 1+2, 2+1, 3}

  for (int i = 4; i <= 11; i++)
    {
      dp[i] = dp[i - 1] + dp[i - 2] + dp[i - 3]; // 점화식
    }

  while(T--)
    {
      int n;
      cin >> n;
      cout << dp[n] << "\\n";
    }
  
}
```

---

## 작은 문제부터 차근차근 해결하기

dp[n]은 정수 n 을 만드는 방법의 수를 뜻한다.

dp[1] = 1 : {1}

dp[2] = 2 : {1+1,2}

dp[3] = 4 : {1+1+1,1+2,2+1,3}

## 점화식 분석

점화식 : 수열의 항 사이의 관계를 나타내는 식

_D_[_n_]=_D_[_n_−1]+_D_[_n_−2]+_D_[_n_−3]

(n-1)을 만들고 +1을 붙인경우

(n-2)을 만들고 +2을 붙인경우

(n-3)을 만들고 +3을 붙인경우

예를 들어 D[4] = D[3] + D[2] + D[1] 을 보면…

- 1+1+1+1 ⇒ (n-1)을 만들고 +1을 붙인경우
- 1+1+2 ⇒ (n-2)을 만들고 +2을 붙인경우
- 1+2+1 ⇒ (n-1)을 만들고 +1을 붙인경우
- 2+1+1 ⇒ (n-1)을 만들고 +1을 붙인경우
- 2+2 ⇒ (n-2)을 만들고 +2을 붙인경우
- 1+3 ⇒ (n-3)을 만들고 +3을 붙인경우
- 3+1 ⇒ (n-1)을 만들고 +1을 붙인경우

D[3] = 4 , D[2] = 2 , D[1] = 1

이렇게 총 7가지가 나온다.
